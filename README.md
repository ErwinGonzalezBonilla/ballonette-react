## ✨ Descripción

Ballonette es una plataforma web enfocada en la presentación de eventos como bodas, cumpleaños, baby showers y eventos corporativos. Ofrecemos una experiencia visual tipo portfolio profesional, donde los usuarios pueden explorar galerías dinámas, subir imágenes en tiempo real y descubrir cada detalle de nuestros trabajos.

Además, Ballonette se especializa en la creación de arreglos personalizados de globos, elegantes composiciones con rosas y servicios de catering, diseñados para transformar cualquier evento en una experiencia única e inolvidable.
